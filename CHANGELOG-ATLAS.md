# Historial De Cambios - IBM i Atlas

## 0.4.0 - 15/07/2026

- Numeracion sincronizada con la distribucion general.
- Sin cambios funcionales ni nuevos prerequisitos para Atlas.

## 0.3.1 - 15/07/2026

- Matriz ampliada de capacidades y artefactos compatibles.
- Casos de uso para RPGLE, SQLRPGLE, DDS, diagnostico, impacto, ILE, QA y rendimiento.
- Prompts listos para copiar y adaptar en proyectos IBM i.
- Ejemplos publicos anonimizados y validacion automatica contra datos internos.

## 0.3.0 - 15/07/2026

- Primera distribucion publica independiente.
- Autodeteccion de VS Code estable, VS Code Insiders y modo portable.
- Orquestador con GPT-5.6 Sol y especialistas con GPT-5.6 Terra.
- Scripts exclusivos de instalacion, actualizacion, diagnostico y desinstalacion.
- Paquete ZIP autocontenido con checksum SHA-256.
